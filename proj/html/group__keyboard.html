enum kbd_key

Enumerated type for specifying the type of key that was pressed in the Keyboard.

Enumerator
NO_KEY 

No key was pressed. May happen when a key is released.

CHAR 

The key corresponding to a character recognized by the program (letter, number, space) was pressed.

CTRL 

Ctrl key was pressed.

ENTER 

Enter key was pressed.

BACK_SPACE 

Backspace key was pressed.

ESC 

Esc key was pressed.

DEL 

Delete key was pressed.

ARROW_UP 

Up arrow key was pressed.

ARROW_DOWN 

Down arrow key was pressed.

ARROW_LEFT 

Left arrow key was pressed.

ARROW_RIGHT 

Right arrow key was pressed.
