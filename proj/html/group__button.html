enum button_state_t

Enumerated type for specifying the state of a button.

Enumerator
BUTTON_NORMAL 

Button is in normal/base state.

BUTTON_HOVERING 

Button is not being pressed but the cursor is hovering it.

BUTTON_PRESSING 

Button is being pressed and the cursor is hovering it.

BUTTON_PRESSING_NOT_HOVERING 

Button is being pressed but the cursor is not hovering it.
