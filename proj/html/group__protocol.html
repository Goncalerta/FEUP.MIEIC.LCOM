Detailed Description

Module dedicated to the application dependent communication protocol between two computers.

enum message_type

Enumerated type for specifying the type of a message.

Enumerator
MSG_READY_TO_PLAY 

Notify player ready to play a new game.

MSG_LEAVE_GAME 

Notify player left the game.

MSG_RANDOM_NUMBER 

Send a random number.

MSG_NEW_ROUND 

Create a new round with a given correct guess.

MSG_START_ROUND 

Notify round start.

MSG_NEW_STROKE 

Notify new stroke being drawn.

MSG_DRAW_ATOM 

Notify new stroke atom drawn.

MSG_UNDO_CANVAS 

Notify canvas undo last stroke.

MSG_REDO_CANVAS 

Notify canvas redo stroke.

MSG_GUESS 

Make a guess for the correct word.

MSG_CLUE 

Show a hint to the word clue at a given position.

MSG_ROUND_WIN 

Notify round won, updating the score.

MSG_GAME_OVER 

Notify round is lost.

MSG_PROGRAM_OPENED 

Notify player opened the program.

◆ protocol_tick()

int protocol_tick ( )

Reacts to a timer interrupt to detect timeouts.

A timeout receiving a message leads to giving up on receiving that message and sending a non-acknowledgment. A timeout waiting for an acknowledgment leads to assuming the connection was lost, giving up on all pending messages and notifying that the other player left the game.

Returns
Return 0 upon success and non-zero otherwise
