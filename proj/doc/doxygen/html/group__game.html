Enumeration Type Documentation

◆ game_state

enum game_state

Enumerated type for specifying the state of the game.

Enumerator
ROUND_UNSTARTED 

Round has not yet started.

ROUND_ONGOING 

Round is ongoing.

GAME_OVER 

Game is over (round lost).

TIMES_UP 

Time is up (waiting for other computer before declaring game over).

ROUND_CORRECT_GUESS 

Correct guess (round win).
