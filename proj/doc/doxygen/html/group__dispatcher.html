◆ handle_other_player_random_number()

int handle_other_player_random_number ( int  random_number)

Reacts to the event that the other player sent the random number to decide starting roles.

Both computers compare the sent and received random numbers before starting the game. The computer that generated the highest number will have the role (role_t) of DRAWER on the first round. In the extremely unlikely case that both players generate the same random number, a new pair is generated and sent between them.

Returns
Return 0 upon success and non-zero otherwise
